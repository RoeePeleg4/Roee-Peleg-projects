# These are the instructions for downloading a YouTube songs to your phone:

1. open this pyhton file saveTube.py located in this directory
2. run this file, choose playlist ot songs, dependent on your needs
3. enter the URL usually this one: https://www.youtube.com/playlist?list=PLp7Wxgm0D8s_rqIw-ZYlMnOW1gSWHZM3l
4. in directory choose this one: D:\music\newSongs, and wait for the download to finish
5. after the download is finished go to onlyNew.py and run it
6. in "enter the directory that all of the songs are in:" enter: D:\music\allSongs
7. in "enter the directory that you want only the new in:" enter: D:\music\newSongs
8. enter 1
9. go to this website: https://getmusic.cc/ and connect to this via the phone app "Offline music"
10. in the app go to import and connect to computer using wifi
11. in the computer click on upload and copy this file
12. run onlyNew.py again 
13. in "enter the directory that all of the songs are in:" enter: D:\music\allSongs
14. in "enter the directory that you want only the new in:" enter: D:\music\newSongs
15. this time choose 2
16. and DONE you can play the songs you like through out your phone, ENJOY!
